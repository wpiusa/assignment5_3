Problem Statement

Create a JavaScript file app.js and include that script file into your index.html file using script tag. In app.js file Create an object Employee. Store name, age and salary and address. Address property should again be a nested object which stores city, state and pincode. Display the value of all properties of Employee object in console.

Employee name, age, salary, city, state and pin code should be displayed
